# 🎮 Echoes of Eternity (Game Website)

## 📌 Description

Echoes of Eternity is a multi-page HTML/CSS website for an action RPG game concept. It showcases game features, gameplay, story, and includes user interaction pages like login and signup.

---

## 🌍 Pages Included

* 🏠 Home
* 📖 About
* 🎮 Gameplay
* ✨ Features
* ⬇️ Download
* ℹ️ Information
* 🔐 Login / Signup
* 💬 Forum
* 🆘 Support

---

## 🚀 Features

* Multi-page navigation system
* Background image design for immersive feel
* Game storyline and lore presentation
* Login & signup UI forms
* Download section
* Styled using CSS animations and hover effects

---

## 🛠️ Tech Stack

* HTML5
* CSS3
